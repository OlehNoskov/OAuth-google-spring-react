# OAuth-google-spring-react
Backend:
- Spring-boot
- Oauth-2.0-Google
- Spring-security


Front-end
- React
- Vite
- React Magma

To launch this you should use version of node like 16.20.2 app execute this command:
- node nvm use 16.20.2
- npm install --force (or --legacy-peer-deps)
- npm run build
- npm run dev
